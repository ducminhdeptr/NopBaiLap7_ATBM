# README - Bài Lập 7

Mô tả

- Đây là bài tập của môn Nhập môn An toàn bảo mật máy tính và truyền tin (ATBMTT).
- Nội dung chính: một trang web đơn giản chứa `index.html`.

Thư mục và file

- `index.html`: file giao diện chính của bài tập.
- `README.md`: hướng dẫn ngắn này.

Cách chạy

- Mở `index.html` trực tiếp bằng trình duyệt (double-click hoặc `Ctrl+O` → chọn file).
- Hoặc chạy server tĩnh tại thư mục này (ví dụ nếu có Python):

```
python -m http.server 8000
# rồi truy cập: http://localhost:8000
```

Yêu cầu

- Trình duyệt web hiện đại (Chrome, Edge, Firefox).
